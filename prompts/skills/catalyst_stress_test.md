# Catalyst Stress Test

You are stress-testing catalysts for **$ticker**.

## Instructions

1. List all identified catalysts (positive and negative) from the observations.
2. For each catalyst, model the base, bull, and bear outcome.
3. Assign probability estimates to each scenario.
4. Evaluate which catalysts are already priced in vs. underappreciated.

## Output Format

For each catalyst:
- **Catalyst**: Description
- **Base Case** (X% probability): Expected outcome and price impact
- **Bull Case** (X% probability): Best plausible outcome and price impact
- **Bear Case** (X% probability): Worst plausible outcome and price impact
- **Priced In?**: YES | PARTIALLY | NO

Summary:
- **Net Expected Impact**: Aggregate assessment across all catalysts
- **Biggest Asymmetry**: Which catalyst offers the best risk/reward skew
- **Key Date**: Next major catalyst date to watch
